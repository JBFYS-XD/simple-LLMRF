# 面向 GGUF 的 LLM 轻量级推理框架设计与实现 —— simple-LLMRF

此项目意为通过对 GGUF 格式的标准的稠密模型进行最简化推理， 来展示基本的推理框架的流程， 并展示一些基础的组件
